# 📖 Overview

Parshuram-CTF is a custom-built Linux Capture The Flag machine that simulates a real-world penetration testing environment. The objective is to enumerate the system, discover hidden clues from a personal portfolio website, gain access to multiple Linux users through SSH, retrieve flags, and finally obtain root access.

The project was developed as part of an MCA Cyber Security assignment to demonstrate Linux administration, web security, user management, SSH authentication, and CTF design.

# 🚩 Flags

Participants must retrieve all flags by progressing through the challenge.

```

/home/<user>/FLAG2.txt
/home/<user>/FLAG3.txt
```
